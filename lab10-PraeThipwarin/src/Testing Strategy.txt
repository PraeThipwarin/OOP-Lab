ฟังก์ชั่น ValidateExpression
- ควรแบ่งตัวเลขและ operator ได้อย่างทุกต้องและจัดเก็บเรียงตามลำดับ
- หากโจทย์ที่รับเข้ามามีตัวอักษรหรือสัญลักษณ์อื่นๆที่ไม่ใช่ operator หรือเป็นบรรทัดเปล่า รวมถึงเรียงลำดับตัวเลขกับ operator ไม่ถูกต้อง ไม่ควรจะเก็บค่าและ return null
- เลขที่เป็นค่าลบ จะเก็บแยกเครื่องหมายลบและตัวเลข

ฟังก์ชั่น CalculateExpression
- ควรคำนวนผลลัพท์ได้อย่างถูกต้องโดยเรียงลำดับการคำนวนจากซ้ายไปขวา
- โจทย์ที่คำนวนไม่ได้ เช่น ขาดตัวเลข หรือoperator ควรจะไม่มีการคำนวน และ return null
- คำนวนเลขที่มากกว่า 16 bits ไม่ได้
- โจทย์ที่ผลลัพท์เป็นทศนิยม ผลลัพท์จะเหลือเพียงเลขไม่มีทศนิยม ไม่ปัด
- / 0 และ % 0 ต้องเป็น null

ฟังก์ชั่น Numeric
- หาก String แปลงเป็นตัวเลขได้ return true
- หาก String แปลงเป็นตัวเลขได้ return false
- ตรวจสอบเลขที่มากกว่า 16 bits ไม่ได้

ฟังก์ชั่น WriteFile
- ข้อมูลที่ส่งไปให้เขียนลงไฟล์และข้อมูลที่อยู่ในไฟล์ควรตรงกันทุกบรรทัด

ฟังก์ชั่น ReadFile
- อ่านไฟล์ที่กำหนดได้อย่างถูกต้องและหากโจทย์ที่มีนั้นถูกต้องจะดึงข้อมูลเขียนลง Result file พร้อมกับผลลัพท์ได้
